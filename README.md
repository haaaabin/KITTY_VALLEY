<a name="Intro"></a>
## 게임 설명

- W / A / S / D 로 이동하고 , 마우스를 사용하여 상호작용합니다.
- 플레이어는 땅을 갈고, 씨앗을 심고, 물을 주어 작물을 재배합니다.
- 획득한 아이템은 툴바와 인벤토리에서 확인할 수 있습니다. 
- 하루가 지나면 작물이 성장하며, 최종적으로 수확하여 판매하여 새로운 씨앗을 구매할 수 있습니다.
